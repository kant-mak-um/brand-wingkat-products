# brand-wingkat-products

**Brand:** WingKat  
**Domain:** https://wingkat.com (apex) and https://www.wingkat.com  
**Category:** Racing catamarans with wing sails (and optional foils)

## Purpose
This repository hosts the public-facing product definitions, documentation, and marketing-ready assets for **WingKat**.

WingKat focuses on **larger performance catamarans** designed for the racing market, built around:
- wing sail rigs (primary identity)
- performance sailing configurations
- optional foil-capable variants (as designs mature)

## Commerce & Memberships
Sales, memberships, and checkout will be handled by **WaterWolf** (private platform).  
This repo is intentionally **public** and contains only publishable product and documentation assets.

## What’s in this repo (initial)
- `/site/` – static website content (landing pages and product pages)
- `/catalog/` – product placeholders (to be populated as kits and accessories are finalized)
- `/LICENSE` and `/LICENSES/` – licensing scaffold (per your standard)

## Status
Pre-launch. Content will expand as production ramps and WingKat kits/accessories are released.
